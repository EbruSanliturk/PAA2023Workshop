Workshop - Introduction to Social Media and Big Data for Migration Studies | PAA 2023, New Orleans, Louisiana
This repository provides materials used in the PAA 2023 workshop on "Introduction to Social Media and Big Data for Migration Studies". This part of the workshop focuses on Twitter data in the studies of International Migration.
